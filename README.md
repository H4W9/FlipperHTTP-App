# FlipperHTTP App
A companion app for FlipperHTTP firmware. 

## Requirements
- Wi-Fi Developer Board, BW16, Raspberry Pi, or ESP32 device flashed with FlipperHTTP v2.2 or higher: https://github.com/jblanked/FlipperHTTP
- 2.4 GHz or 5 GHz Wi-Fi access point
